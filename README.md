# NestJS & Angular Mastery Guide v20

Una guía interactiva y moderna diseñada para llevarte de cero a Senior en el desarrollo Fullstack.

## 🚀 Tecnologías
- **Core**: Vite.js + TypeScript
- **Frontend Concepts**: Angular v19 (Signals, Standalone, Resource API, Defer)
- **Backend Concepts**: NestJS (SOLID, Guards, Interceptors, DTOs)
- **Arquitectura**: SOLID, POO, KISS

## 📦 Instalación

```bash
npm install
```

## 🛠️ Desarrollo

```bash
npm run dev
```

## 🚀 Despliegue en GitHub Pages

Para publicar esta guía en tu propio GitHub Pages:

1. Asegúrate de haber subido tu código a un repositorio de GitHub.
2. Ejecuta el comando de despliegue:

```bash
npm run deploy
```

El proyecto se compilará automáticamente y se subirá a la rama `gh-pages`.

## 🏛️ Arquitectura del Proyecto

Este proyecto sigue los principios **SOLID** y **POO**:
- **Sidebar.ts**: Componente encargado de la navegación.
- **ContentArea.ts**: Gestor del renderizado dinámico.
- **MarkdownParser.ts**: Utilidad para el procesamiento de texto.
- **models.ts**: Definiciones de tipos e interfaces.
